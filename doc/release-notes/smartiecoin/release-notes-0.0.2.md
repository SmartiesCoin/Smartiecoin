# Smartiecoin Core version v0.0.2

Smartiecoin Core `v0.0.2` is a maintenance release focused on wallet UX and bootstrap reliability.

## Highlights

- Wallet ticker corrected from `SMTY` to `SMT` in GUI/unit labels and fee unit output.
- Dark mode fixed in Qt wallet by applying a stable native Qt dark palette.
- Mainnet bootstrap seed list now includes the primary node `207.180.230.125`.

## Upgrade notes

- No consensus or transaction format changes are introduced in this release.
- Existing wallets remain compatible.
